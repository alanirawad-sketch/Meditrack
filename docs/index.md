# Meditrack

Meditrack is a simple command-line tool that helps users track their daily medications.

## Problem
Many people forget to take their medication or forget if they already took it.

## Solution
Meditrack allows users to:
- Add medications
- List daily medications
- Mark medications as taken

## Target Users
- Students
- Elderly people
- Caregivers
- Anyone taking daily medication

## Pricing
- Free for personal use
- Future: Pro version with reminders
